# **App Name**: BhashaSetu

## Core Features:

- Landing Page: Simple landing page with a clear call to action to start learning.
- Language Selection: Allow users to select an Indian language to learn from a list of supported languages.
- Dynamic Lesson Generation: Generate learning content (translations, examples) dynamically using the Gemini API.
- Speech to Text Practice: Use Deepgram STT, convert speech to text, and give users feedback via Gemini's evaluation tool on their pronunciation and grammar.
- Gamified Learning: Implement a points, badges, and level progression system to encourage learning.
- Progress Tracking: Track user progress locally (using anonymous Firebase user) and display it on a dashboard.
- Mini-Games: Engage users with word matching and quizzes for enhanced retention.

## Style Guidelines:

- Primary color: Saturated purple (#A020F0) to convey energy and excitement for language learning.
- Background color: Light purple (#F0E5FF) creates a soft, inviting backdrop.
- Accent color: Cyan (#00FFFF) for interactive elements to draw attention and provide a fresh, modern look.
- Body and headline font: 'PT Sans' for a modern, accessible, and easily readable interface.
- Use simple, clean icons to represent different lesson categories and gamification elements.
- Mobile-first, responsive layout that adapts to different screen sizes.
- Subtle animations and micro-interactions to provide feedback and celebrate user achievements.
